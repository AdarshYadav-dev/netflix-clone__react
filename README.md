## Features
- User Signup, Login and Logout using Firebase  
- Fetch movies from **OMDb API**
- Play movie trailers via **YouTube Data API v3**
- Dynamic, scrollable movie galleries like “Popular”, “Blockbuster”, etc.
- Responsive navigation bar with dark-on-scroll effect

## Tech Stack
- **React.js** (frontend)
- **Firebase** (Auth & Firestore)
- **OMDb API** (movie data)
- **YouTube Data API v3** (dynamic trailer embeds)
- **Vite** (dev server)
